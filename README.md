# DentDesk Landing

Static landing page for DentDesk.

## Run

Open [index.html](./index.html) directly in a browser, or serve the folder with any static file server.
or you can run:
<code> python3 -m http.server 8888 </code>

## Notes

- The consultation forms save submitted leads into browser `localStorage` under `dentdesk-demo-leads`.
- The page uses Google Fonts for the final typography pair.
